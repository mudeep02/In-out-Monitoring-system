## 📌 Project Overview

The **AI In-Out Monitoring System** is a real-time computer vision application that automatically detects, tracks, and counts people entering and exiting a designated area using advanced object detection models.

The system processes live video streams or recorded footage to monitor occupancy, making it suitable for offices, educational institutions, shopping malls, hospitals, and public spaces.

---

## 🚀 Features

✅ Real-time person detection using YOLOv8 and MobileNet-SSD

✅ Automatic In and Out counting across a virtual boundary

✅ Multi-person tracking with high detection accuracy

✅ Live video processing using OpenCV

✅ Supports CCTV footage, webcam, and recorded videos

✅ Displays real-time occupancy statistics

✅ Reduces manual monitoring and improves operational efficiency
